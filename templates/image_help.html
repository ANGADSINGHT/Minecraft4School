How to Use Powershell

Step 1: Open the Windows Apps Menu
Click on the Windows icon at the bottom left (or press the Windows key) to access a list of apps.
Note: if on Windows 11, just search for Powershell with the search bar
Step 2: Scroll to the bottom
Scroll down to find the Windows Powershell app in the list.
Step 3: Open Powershell
Click on Windows Powershell to open it. You should see a blue or black window appear.
Step 4: Paste the Script
Copy the script from the website and right-click or press Ctrl+V to paste it into Powershell. Press Enter to run.
